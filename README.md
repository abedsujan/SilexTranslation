SilexTranslation
======================

Installing
----------

``` bash
$ git clone
```

``` bash
$ composer install
```

Browsing the Demo Application
-----------------------------

To see a real-live Silex page in action, start the PHP built-in web server with
command:

.. code-block:: console

    $ cd path/to/install
    $ php -S localhost:8888 -t web/ web/index_dev.php

Then, browse to http://localhost:8888/.
